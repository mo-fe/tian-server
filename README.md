# Tian 服务端

## Installation

```bash
$ npm install
```

## 账号相关配置信息,统一写在.env配置文件里面

```
$ cp env.example .env
$ vi .env
```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
